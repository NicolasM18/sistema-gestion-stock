Sistema de Gestión de Stock

Este proyecto es una aplicación web Full-Stack diseñada para la administración y el control eficiente de inventarios. Fue desarrollado como proyecto personal para aplicar y consolidar conocimientos en arquitectura de software, backend y frontend.

--Tecnologías Utilizadas

* **Backend:** Python, Django
* **Frontend:** JavaScript, HTML, CSS, Bootstrap
* **Base de Datos:** SQLite (o la que estés usando)

--Funcionalidades Principales

* Registro, edición y eliminación de productos en tiempo real.
* Control de cantidades y alertas de stock bajo.
* Interfaz de usuario dinámica y responsiva creada con Bootstrap y JS.
* Arquitectura escalable basada en el patrón MVT de Django.
